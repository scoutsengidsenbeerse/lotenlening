# Scouts en Gidsen Beerse – Lotenlening 2026

Statische website voor de lotenlening van VZW Scouts en Gidsen Beerse.

## Structuur

```
index.html          ← Hoofdpagina (alles op één pagina)
css/
  style.css         ← Alle stijlen
js/
  script.js         ← Navigatie, accordion, animaties
README.md
```

## Technologie

- Puur HTML5, CSS3 en vanilla JavaScript
- Geen frameworks, geen build tools, geen dependencies
- Google Fonts (Playfair Display + Inter) via CDN
- Volledig responsief (mobiel, tablet, desktop)
